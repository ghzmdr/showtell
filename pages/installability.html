Installability

Native apps are installable so they persist on the device, have no URL bar, have an icon you can tap on.
We can have that also.
Shipping in all mayor browsers, is support for installable web apps.
These need to declare their properties in a manifest.json, install a service worker and a few other still changing parameters.
Here the criteria to be met so that your app will get the 'Add To Homescreen' banner. (For now chrome on android only).

This feature, even better if combined with others, lowers dramatically the difference the user percieves between an application and a website.

On iOS this can be simulated by <meta> tags, but these have to be used wisely, see this.
